# Repository Docs

Durable MiniPainterHub documentation lives in `ObsidianVault/`, starting at
`ObsidianVault/00 Start Here/Project Index.md`.

This `docs/` folder is for historical or task-local artifacts that should not
be treated as source-of-truth project guidance. If a note becomes durable
architecture, process, design, deployment, or testing guidance, move or rewrite
it into the smallest matching vault folder and update the vault index when it
should be discoverable from the root.
