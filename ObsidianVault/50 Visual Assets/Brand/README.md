# MiniPainterHub Brand Assets

This folder contains the proposed MiniPainterHub logo system as editable SVG artwork.

## Asset inventory

- `brand-board.svg` — overview of the logo system, approved colors, and scale checks.
- `logo-mark.svg` — primary square emblem for avatars, app marks, and compact placements.
- `logo-lockup.svg` — primary horizontal lockup for light surfaces.
- `logo-lockup-reversed.svg` — horizontal lockup for dark surfaces.
- `favicon.svg` — simplified small-scale mark for browser and launcher contexts.

## Source and intent

The artwork is original and was designed specifically for MiniPainterHub. It uses the existing product palette and visual language rather than introducing a separate brand theme.

The emblem combines three ideas:

1. a miniature display figure and plinth for the craft,
2. a fine brush for painting,
3. a circular ring with nodes for the community hub.

## Export guidance

- Keep SVG as the source of truth.
- Export PNG only when a destination cannot consume SVG.
- Preserve the SVG `viewBox` and aspect ratio.
- Do not crop the rounded-square mark or recolor individual components.
- Use the reversed lockup only on dark, low-detail surfaces.
- Use the dedicated favicon instead of shrinking the full lockup.

Implementation and spacing rules live in [Brand Identity](<../../40 Design/Brand Identity.md>).
