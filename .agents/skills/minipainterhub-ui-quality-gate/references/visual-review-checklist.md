# MiniPainterHub Visual Review Checklist

Check each reviewed screen for:
- balanced layout and appropriate content width
- consistent spacing and alignment
- clear hierarchy between headline, body, and actions
- coherent button and form control sizing
- no clipping, overlap, truncation, or accidental dead space
- responsive behavior at desktop and narrow widths
- visible focus styles and readable contrast
- product-level polish, not just technical correctness

If a screenshot looks obviously awkward, continue iterating before closing the task.
The default response to an in-scope issue is: fix it, rerun the browser review, and verify the fix visually.
