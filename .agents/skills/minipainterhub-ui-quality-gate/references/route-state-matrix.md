# MiniPainterHub Route and State Matrix

Shared shell and community:
- `/`
- `/posts/all`
- `/posts/top`
- `/search`
- `/feed/following`
- `/messages`
- `/profile`

Auth:
- `/login`
- `/register`

Post/content:
- `/posts/new`
- seeded post details
- `/posts/mine`
- public profile
- `/connections`

Admin:
- `/admin/moderation`
- `/admin/reports`
- `/admin/audit`
- `/admin/suspensions`

Required shell states:
- desktop panel open
- desktop panel collapsed
- mobile personal panel open

Required state variants when relevant:
- empty
- populated
- auth-gated
- validation or error state
